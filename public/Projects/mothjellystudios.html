Studio website • Performance-first • Clear product presentation

The public website for Moth Jelly Studios. I designed and built it end-to-end to be fast, readable, and structured around what matters most: showcasing shipped work and making it easy to understand what the studio builds.

Design & Engineering Notes

What I optimized for and how it’s structured.

Information Architecture

  • Designed navigation to route visitors to shipped work fast
  • Prioritized scannability: short sections, strong headings, minimal clutter
  • Made “what we build” obvious within seconds

Performance

  • Optimized images for fast loads and good visual clarity
  • Kept UI lightweight (no heavy frameworks required)
  • Styled components for readability on mobile first

Reusable Layout System

  • Built consistent “sections” and “cards” so pages are easy to extend
  • Created predictable spacing and typography rules
  • Used a clean component-like pattern in plain HTML/CSS

What I’d Improve Next

  • Add lightweight analytics (privacy-friendly) to learn what visitors click
  • More case-study depth per game (technical + design notes)
  • Optional presskit downloads per project
